Set of 10, 24 dimensional, 3 mode coupled, potential energy surfaces.

Molecules included:
  - Butadiene
  - DMSO
  - Ethylamine
  - Ethylene Glycol
  - Nitroethane
  - Propanal
  - Pyrazine
  - Pyrrole
  - Thioacetone
  - Vinylformamide

Datafiles are found in the test and train folders and have the following format:
The first line is the reference value for all modes (0.0) and the energy of this reference structure.
All following lines are structured as
  Displaced dimensions : Displacements along each dimension : Energy

Also included is materials used to create the training set in MIDASCPP with calculations performed using XTB.
Molecular structures used are in molecules.
Example calculation with midas input file and necessary interface files can be found in example.

Links:
